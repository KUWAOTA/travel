# JICA 青年海外協力隊 検討プロジェクト

## 概要
JICA青年海外協力隊への応募を検討中。意思決定に必要な情報収集と関係者への相談を段階的に進める。

## タスク管理
- **ボード**: [BOARD.md](BOARD.md) — 全タスクの一覧とステータス
- **各タスク詳細**: [tasks/](tasks/) 配下の個別ファイル

## このプロジェクトでの作業ルール

### タスクを進める時
1. ユーザーが「1-2をやりたい」等と言ったら、該当する `tasks/` のファイルを開いて内容を確認
2. タスク内のチェックリストに沿って作業を進める
3. 完了したらタスクファイルのステータスを更新し、BOARD.mdも更新する
4. 結果・メモはタスクファイルの「結果メモ」「メモ欄」に記録する

### ステータスの表記
- ⬜ 未着手
- 🔄 進行中
- ✅ 完了
- ⏸️ 保留

### 相談の壁打ち
ユーザーが「父親に相談したい」「彼女と話したい」と言った場合:
- 該当タスクの「聞きたいこと」をベースに会話のシミュレーション・準備をサポート
- 相談後の結果をタスクファイルに記録する手伝い

### 案件検討
- 案件データは `prompt/A241.md` と `prompt/G151.md` に格納
- ユーザーのスペック: Java/Android 4年, Python 2年, 京大電電→NAIST情報修士, 村田5年目
- 希望地域: 南米・アフリカ・パプアニューギニア（昆虫層の多い熱帯）

- 2026/4/15: 春募集締切（今年度は応募しない）
- 2026/4/18-19: JICA祭り（情報収集のターニングポイント）
