# 2-1: グローバル人事部への質問整理

**ステータス:** ⬜ 未着手
**期限:** 4月中
**戻る:** [← ボード](../BOARD.md)

## 目的
村田製作所のグローバル人事部に確認すべき質問を整理する。意思決定マトリクスの選択肢1（現職参加）と選択肢3（OTP）の実現可能性を明らかにする。

## 質問リスト

### 現職参加について
- [ ] 村田製作所にJICA青年海外協力隊の現職参加制度はあるか？
- [ ] 過去に利用した社員はいるか？
- [ ] 制度がない場合、休職扱いで対応できるか？
- [ ] 休職の場合、籍は残るのか？復帰後のポジションは？
- [ ] 人事部としてこの話を課長に先に通すべきか、それとも課長経由で相談すべきか（根回しの順番）

### OTP（海外若手派遣）について
- [ ] OTPの次回募集時期・対象条件の最新情報
- [ ] イタリア子会社への出向の見込み・タイムライン
- [ ] OTPとJICAを組み合わせる（OTP後にJICA等）ことの現実性

### キャリア全般
- [ ] 5年目のエンジニアが2年離れた場合の復帰パス
- [ ] グローバル人材育成の観点からJICA経験をどう評価するか

## アプローチ方法
- [ ] まずは人事部の誰に聞くべきか特定する
- [ ] メール or 面談のどちらが適切か判断
- [ ] 質問を送る前に信頼できる先輩に相談するか検討
